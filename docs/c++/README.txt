SITE WEB
--------
https://isocpp.org/


USAGE DE G++
------------
    `g++ myscript.cpp`
    ------------------
        Compile le fichier source myscript.cpp dans l'executable a.out.
    
    `g++ -o main myscript.cpp`
    --------------------------
        Compile le fichier source myscript.cpp dans l'executable main.


EXEMPLES
--------

    ./helloworld.cpp
    ----------------
        Programme d'affichage du message "Hello, World !" sur la sortie standard

    ./algorithmes/booleens.cpp
    --------------------------
        Programme qui recapitule la declaration et la manipulation des objets de
        type booleens et les expressions booleennes.

    ./algorithmes/caracteres.cpp
    ----------------------------
        Programme qui recapitule les fondamentaux sur les chaines de caracteres 
        en C++. Contient en commentaire les liens vers les references utiles des
        bibliotheques utilisees.

    ./algorithmes/command_line_file_io.cpp [TO FIX (Ubuntu), OK (Windows)]
    --------------------------------------
        Programme qui accepte des arguments passes a l'appel, en ecrit un dans 
        un fichier texte et qui dort pendant dix secondes avant de supprimer le
        fichier.

    ./algorithmes/event_handler.cpp [TO FIX (Ubuntu), OK (Windows)]
    -------------------------------
        Programme qui reagit a un evenement provoque par l'utilisateur. Le 
        programme compte jusqu'a l'appui sur une touche par l'agent, alors le
        programme termine son execution.
    
    ./algorithmes/pointeurs.cpp
    ---------------------------
        Programme qui recapitule les fondamentaux sur les pointeurs et leurs 
        usages en langage C++.

    ./algorithmes/sleeper.cpp [TO FIX (Ubuntu), OK (Windows)]
    -------------------------
        Programme qui dort pendant une duree determinee.
